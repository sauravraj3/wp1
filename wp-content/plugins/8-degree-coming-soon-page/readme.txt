You must activate the plugin and go for a few more clicks, you will have your website's coming soon page created. = Free Version Features: =

1. <strong>Option to configure background with less complexibility:</strong> The plugin is highly configurable as it has options to easy configuration. You can choose background color and set it easily or background image from pre available images.You can also upload background image of your own.

2. <strong>Text configurations option:</strong> It has simple text configuration options as well. You can configure headline , main text and footer credit easily. You can choose font color for the text anywhere in the page.

3. <strong>Email subscription:</strong> The plugin has better email subscription plans like option with backend email listing, csv export option, customizing “Thank You” message of subscription form etc.

4. <strong>Social Icons:</strong> The plugin is also social network friendly. It has option to configure links in fixed 6-8  icons and options to choose social icons color and background color.

5. <strong>Countdown display for site launch:</strong> The plugin offers two different timer layouts which allows you to display countdown time for your site launch as per your choice.

6. <strong>Mobile responsive and translation ready:</strong> The plugin is fully responsive that has good performance on any device. Besides, it is translation ready which can be translated into any language of the world.

== Installation ==
1. Unzip 8degree-coming-soon-page.zip
1. Upload all the files to the /wp-content/plugins/8degree-coming-soon-page
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. For customizing the plugin's settings, click on Coming Soon option menu in WordPress left admin menu.


== Frequently Asked Questions ==
= What does this plugin do? =
This plugin provides the ability to display a under construction page with a timer for a site launch date with various configurable backend options.

= Do I need to design the coming soon page? = 
You won't need to take any burden regarding the design as since we have already provided various configuration options such as background color/image, timer layout, font color, etc using which you can configure the design as per your requirement.

Once you install the plugin , you can check some general documentation about how to use the plugin in the "How to use" panel of the plugin's settings page.
